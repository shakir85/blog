## Blog
This repo hosts my personal blog code built using [Hugo](https://gohugo.io/) static site generator.

## How to run
- Install git 
- Install [go](https://go.dev/doc/install)
- Install [hugo](https://gohugo.io/installation/)
- Run `hugo server`
